Nossos canais

Seja para tirar uma dúvida relacionada, se inteirar das novidades ou conhecer a comunidade de PHP de Minas Gerais, veja os canais nos quais podemos ser encontrados:

  • Slack

    Este é o local onde as discussões mais cotidianas acontecem. Amizades são fortalecidas aqui, e eventualmente, alguns assuntos polêmicos aparecem

  • Lista de emails

    Onde o grupo nasceu. A interação é pouca, mas ainda acontece.

  • MeetUp.com

    Na plataforma do MeetUp nós divulgamos todos os eventos do grupo. É o local para acompanhar quando serão os próximos eventos e encontros.

  • Twitter

    Algumas novidades são postadas aqui.

  • Facebook

    Temos uma página e um grupo. O grupo já foi mais ativo, mas acabou perdendo pro Slack.

  • GitHub

    Alguns projetos de múltiplas mãos que nasceram sob o teto do PHPMG ficam organizados no GitHub - incluindo este próprio site!
